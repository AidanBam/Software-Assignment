How to play

This is a game to help you study, roll the dice for a number, if you go over a INFO tile, you will recive information about possible future questions, try to remember this, if you land on a question tile you will have to answer the question right over wise you will go back 5 tiles, if you get the question right, you will gain points.

Once you reach 100 points you will be able to attempt the quiz, if you beat the quiz you win, if you fail you will be reset to 0 points!

Once you have read all of this press continue below to start the game!
